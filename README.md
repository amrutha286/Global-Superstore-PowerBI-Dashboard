# Global-Superstore-PowerBI-Dashboard
This Power BI project analyzes the Global Superstore dataset through interactive visualizations of sales, profit, and market trends.

Interactive Power BI dashboard for Global Superstore sales analysis, featuring KPIs like total sales (12.6M), average sales (246), profit margins, and market breakdowns by region, category, and time. Users explore bar charts, line graphs, pie charts, and treemaps for quick insights into performance trends.


This project transforms the Global Superstore dataset into a dynamic dashboard showcasing sales performance across markets, categories, and time periods. Visualizations include bar charts for market-wise sales, line graphs for quarterly/yearly trends, and pie charts for category/sub-category distributions. The purple-themed layout with KPIs provides instant insights into global operations.


Key Features

•	Sales & Profit KPIs: Displays total sales (12.6M), average sales (246), total profit (1.47M), and quantity (17.8K) for quick performance snapshots.
•	Market Analysis: Stacked bar charts break down sales by market, highlighting top performers like the US.
•	Temporal Trends: Line graphs visualize sales and profit by year/quarter/month, revealing growth patterns from 2014-2018.
•	Category Breakdowns: Pie charts and treemaps show sales/profit by category (e.g., Technology, Furniture) and sub-category distributions.
•	Interactive Elements: Supports filters by segment, region, and date; enables drill-downs for deeper exploration.

Technical Details

Built using Power BI Desktop with the Global Superstore dataset, featuring DAX measures for KPIs and slicers for interactivity. The dashboard spans multiple pages, including a high-level summary and detailed views like profit by sub-category. Optimized for stakeholder presentations, with tooltips and cross-filtering for data-driven decisions on underperforming areas and growth opportunities.
